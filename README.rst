Requirements
------------

* Python 3.10 
* Poetry
* Docker containers (if you want to use remote containers and Visual Studio Code)


Installation
------------

You can install *Banking Kata Python* via pip_ from PyPI_:

.. code:: console

   $ pip install banking-kata-python
